
# Land Parcel / Boundary Survey & Plot Visualizer

## Goal
Compute polygon vertices from boundary lines, check closure error, calculate area, and display a 2D plot.

## Features
- Computes polygon coordinates
- Checks closure error
- Calculates area using Shoelace formula
- Displays 2D plot using Matplotlib

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run program:
   python main.py
